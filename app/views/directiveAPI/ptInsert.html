ptInsert

Note:

ptInsert is a directive that you can use this to insert record to database simply...new object must named 'newData'


Arguments:

Param Type Details
data object [Required]

the record which insert into database

service String [Required]

the operate service
flag 1 or 0 [Optional]

If you set this param to 0,you will use the standard way to send http request,or to 1 that you will use post method to do every operation,default :0
redirect object [Optional]

redirect url and param

Usage:

Controller:

var testController = angular.module('testController',[])
                testController.controller('insertController',function($scope){
                $scope.insertData = {
                'id':'1',
                'name': 'insert'
                }
                $scope.insertRedirect={
                'url':'#',
                'param':'1'
                }
                })

Service:

testService.factory('insertService',function($resource){
                return $resource("#");
                })

Html:

<div ng-controller="insertController">
            <button pt-insert data="insertData" service="insertService" flag="0" redirect="insertRedirect">add
            </button>
            </div>
