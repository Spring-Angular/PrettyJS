ptInsert

Note:

ptInsert is a directive that you can use this to insert record to database simply...


Arguments:

Param Type Details
data object

the record which inset into database

service String

the operate service
flag 1 or 0

0 :(add: get method)/1 : (add: get method)
redirect object

redirect url and param

Usage:

The normalization process is as follows:

Controller:


            
var testController = angular.module('testController',[])
                testController.controller('insertController',function($scope){
                $scope.insertData = {
                'id':'1',
                'name': 'insert'
                }
                $scope.insertRedirect={
                'url':'#',
                'param':'1'
                }
                })

Service:


            
testService.factory('insertService',function($resource){
                return $resource("#");
                })

Html:


            


                <div ng-controller="insertController">
                <button pt-insert data="insertData" service="insertService" flag="0" redirect="insertRedirect">add
                </button>
                </div>
