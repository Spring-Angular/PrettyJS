ptUpdate

Note:

ptUpdate is a directive that you can use this to update record to database simply,if you use ngResource,you should generate a update method(http method:put) in your service,because ngResource has no default update method


Arguments:

Param Type Details
data Object [Required]

The object of the update
service Object [Required]

The service of the update

redirect Object [Optional]

If you want to redirect after update,you should set redirect'url in your controller,than use this param to pass url
flag String [Optional]

If you set this param to 0,you will use the standard way to send http request,or to 1 that you will use post method to do every operation,default :0

Usage:

The HTML code is as follows:

            
                <button type="button" class="btn btn-primary btn-flat pull-right" style="margin-right: 100px;" ng-disabled="!vendorForm.$valid" update data="newData" service="catalogOperationService" redirect="redirect">Submit

The Service code is as follows:

            catalogService.factory('catalogOperationService', function ($resource,baseUrlService) {
                return $resource(baseUrlService.baseUrl+'catalogd/:id',{},{
                update: {method:'put'}
                });
                });
