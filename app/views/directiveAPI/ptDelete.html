ptDelete

Note:

ptDelete is a directive that you can use this to delete record from database simply...list collection must named 'dataList'


Arguments:

Param Type Details
data object [Required]

the record which delete from list collection

service String [Required]

the operate service
idx Object [Required]

pass a index in list collection of delete object,is usually $index
msg object [Required]

get a name or other attrs to use into show modal
flag String [Optional]

If you set this param to 0,you will use the standard way to send http request,or to 1 that you will use post method to do every operation,default :0

Usage:

Html:

<i class="fa fa-trash-o"  style="margin-left: 5px;cursor:pointer;" data-original-title="Delete Catalog" data-toggle="tooltip" data-placement="top" tool-tips delete msg="catalog.catalogVO.name" data="catalog" idx="$index" service="catalogOperationService">
